🌿 The Green Edit

A beautifully designed, beginner-friendly web application that helps users explore, choose, and care for indoor plants. This project combines aesthetic UI with practical guidance for plant lovers living in urban spaces.

Overview:

The Green Edit is a static front-end website focused on:

Helping users discover plants suitable for different spaces
Educating beginners about plant care
Showcasing benefits of having indoor plants
Providing a visually appealing and calming user experience
